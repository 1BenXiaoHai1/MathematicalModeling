下面是一个流程图：

​```mermaid
flowchart TD
    A[输入：原始数据集] --> B[数据预处理]
    B --> B1[处理缺失值/异常值]
    B --> B2[编码分类特征]
    B --> B3[划分训练集/测试集]
    B --> C[核心步骤1：Bootstrap采样生成子数据集]
    C --> D{对每个子数据集}
    D --> E[核心步骤2：特征随机选择]
    E --> E1[节点分裂时随机选k个特征<br>k=√总特征数]
    E1 --> F[核心步骤3：构建决策树]
    F --> F1[基于信息增益/基尼系数<br>选择最优分裂]
    F1 --> F2[递归分裂至停止条件]
    F2 --> F3[生成完整决策树]
    F3 --> G{是否生成足够树？}
    G -- 否 --> C
    G -- 是 --> H[核心步骤4：模型集成]
    H --> H1[分类：多数投票]
    H --> H2[回归：取平均值]
    H1 & H2 --> I[输出最终预测结果]
    I --> J[模型评估]
    J --> J1[分类指标]
    J --> J2[回归指标]
```
